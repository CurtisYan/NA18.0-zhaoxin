动画时序重构
将“滑出”拆分为两阶段：前 1/3 慢速（exiting_slow）、后 2/3 快速并同时触发下一段滑入（exiting_fast_entering）。
新的状态机：idle → exiting_slow → exiting_fast_entering → idle，严格用定时器管理各阶段切换，保障节奏稳定。
缓动与衔接优化
慢段使用 ease-in（结尾更快），快段使用丝滑的 ease-out，使两段交界时速度更接近，减少“突然变快”的突兀感。
在快段继续沿用每行错峰（stagger），让慢段到快段的过渡更自然。
第二段进入的速度与柔化
新增 enterDelayMs（默认 30ms）：第二段开始滑入的小延迟，避免同帧同时加速。
新增 enterFastFactor（默认 0.8）：第二段滑入相对“快速阶段”的时长比例（<1 更快），可快速提升进入速度。
为第二段叠加轻微淡入（opacity 0.9 → 1），缓解速度变化的视觉突变。
可配置与可调参设计
保留并兼容原有参数：intervalMs、pauseMs、durationMs、enterDurationMs、staggerMs、enterEasing、exitEasing、className 等。
新增可控参数：enterDelayMs、enterFastFactor；后续可轻易扩展 slowFraction（慢段比例）等更细粒度调参点。
逐行错峰与高度测量
慢段与快段均考虑 per-line stagger；慢段含错峰总时长，快段延续同样的 per-line delay，保证视觉连续性。
维持原有高度测量策略（容器/单行测量）以防止抖动，确保布局稳定。
代码与结构层面
计时器精细拆分：采用 tSlow（1/3）与 tFast（2/3）并合并错峰总时长控制阶段切换点。
明确的阶段内样式控制：当前行 transform 从 0%→33.333%（慢）→100%（快）；下一行在快段从 -100%→0%，并伴随淡入。
修复一次编辑引入的语法错误（phase 初始值），保证类型与语法正确。
视觉/体验效果
解决“第二段在 1/3 时开始滑入且速度与后 2/3 一样快”的核心诉求。
针对“慢到快很生硬”的反馈，连续优化缓动曲线、错峰延续、进入延迟与淡入，使衔接更顺滑。
